Command Usage

This page lists the available commands in the server. The parameters are as follows:
  1. Plain-text is MANDITORY
  2. <This> text is MANDITORY, and denotes a parameter such as player name.
  3. [This] text is an optional parameter you can use when calling the command.

Worlds

Commands and permissions may depend on the world you are currently in. The worlds are:
  1. Hasumu - The HArd-SUrvival-MUltiplayer World (Given a new name when reset, currently 'Scippe')
  2. Sigby - The member survival build world
  3. Crafterville - Survival Plot world
  4. Rhyolite - Creative world

Commands by Rank

Aspirant:

All new players joing the server as Aspirants. You will have access only to the Hasumu world.
    Commands
  1. /spawn
  2. /dynmap webregister
  3. /ticket <message>
  4. /me <message>
  5. /tell <player> <message>
    Information
  1. /ontime [top]
  2. /who
  3. /isbanned <player>
  4. /motd
  5. /rules
  6. /ontime [top]

Survivalist:

After 2hr of logged time a player will become a Survivalist. Players gain access to all other worlds, and may purchase plots there.
    Commands
  1. /rg info [plot]
  2. /rg list
  3. /rg addmember <region> <player>
  4. /rg addowner <region> <player>
  5. /sethome
  6. /home
  7. /nick <nickname>
    Plots
  1. /plot <claim>
  2. /plot visit <player>
  3. /plot add <player>
  4. /plot remove <player>
  5. /plot deny <player>
  6. /plot merge [direction]

Adept:

After 2 days of online time a player will become an Adept. They are experienced players who have invested much time in the server. They may now use italics and bold formatting in their nicknames, and may capture mobs with eggs free of charge.

Guru:

After 7 days of online time a player will become a Guru. They may use all formatting in chat as well as the ability to hide themselves on the dynmap.
  1. /dynmap <hide|show>

Ensign

Ensign is the first tier for moderators. If you'd like to become an Ensign, you must possess the required qualifications.
    Qualifications
  1. Be at least a Survivalist
  2. Know and understand the server rules
  3. Have a mastery of the server commands & usage
    Commands
  1. /broadcast <message>
  2. /kick <player> [reason]
  3. /nick <player> <nick>
  4. /ncp lag
  5. /pex user <player>
  6. /bookgive <player> <book>
    Markers (Use this reference)
  1. /dmarker add <name> icon:<icon-id>
  2. /dmarker delete <name>
    Tickets
  1. /tickets
  2. /helptickets
  3. /taketicket <#>
  4. /closeticket <#>
  5. /replyticket <#>
  6. /delticket <#>
    WorldGuard (Same as above, but can view regions)
  1. /rg info [region]
  2. /rg list

Lieutenant

Lieutenants can add any player to any region as an owner or a member.
    Commands
  1. /ban <player> [reason]
  2. /unban <player>
  3. /whois <player>
  4. /dynmap <show|hide> [player]
  5. /tm bc <message>
  6. /slap <player>
    Warps
  1. /warps <list|del|info>
  2. /setwarp <warp>
